# How to Use
## Setup @api/client
 `cd api-client`
 Install dependencies
 `npm install` or `yarn`
 Build
 `npm run build` or `yarn build`
 Link
 `yarn link`

 ## Link frontend & admin project folders
 Navigate into respective folder
 `cd admin` or `cd frontend`
 Link
 `yarn link "api-client"`
